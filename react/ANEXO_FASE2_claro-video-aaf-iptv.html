### 2.1 Dead Code (javascript:S1763) — 118 instancias

**Descripción:** Código que nunca se ejecuta porque aparece después de `return`, `throw`, `break` o `continue`.


**Acción:** Eliminar código muerto o mover antes del `return`.

### 2.2 Callbacks sin Return (javascript:S3796) — 26 BLOCKER

**Descripción:** Callbacks en métodos como `map()`, `filter()`, `find()` que no retornan valor explícitamente. Esto causa que el callback retorne `undefined` en lugar del valor esperado.

**Archivos críticos:**
- `src/utils/Search.js` — 23 ocurrencias (concentración crítica)
- `src/requests/loader.js` — 1 ocurrencia (línea 473)
- `src/utils/Npvr.js` — ocurrencias adicionales
- `src/utils/Utils.js` — 3 ocurrencias

**Impacto en producción:** Arrays de resultados potencialmente llenos de `undefined`. Búsquedas retornando resultados vacíos o incorrectos.

**Corrección:**
```javascript
// ❌ BEFORE (BLOCKER)
array.map(item => {
  processItem(item); // falta return
});

// ✅ AFTER
array.map(item => {
  return processItem(item);
});
// O versión concisa: array.map(item => processItem(item));
```

```mermaid
    User->>LoginUI: Ingresa credenciales
    LoginUI->>AuthActions: dispatch(fetchIsLoggedIn / fetchSmartCardLogin)
    AuthActions->>Redux: dispatch loading state
    AuthActions->>AAFAuth: Auth.login(credentials)
    AAFAuth->>RequestMgr: POST /user/login (vía RequestManager)

    alt Login exitoso
        RequestMgr->>API: POST /user/login
        API-->>RequestMgr: 200 OK + JWT Token
        RequestMgr-->>AAFAuth: token response
        AAFAuth->>Storage: Persiste token (DeviceStorage)
        AAFAuth-->>AuthActions: Success
        AuthActions->>Redux: dispatch(updateUserState)
        Redux-->>LoginUI: state.user actualizado
        LoginUI-->>User: Redirige a Home
    else Token expirado — Retry con refresh
        RequestMgr->>API: Fetch falla (401)
        RequestMgr->>RequestMgr: refreshToken() vía requests/loader.js
        RequestMgr->>API: POST /user/login (retry)
        API-->>RequestMgr: Nuevo token
        RequestMgr-->>AAFAuth: Success tras retry
    else Credenciales inválidas
        API-->>RequestMgr: 401 Unauthorized
        RequestMgr-->>AAFAuth: Error parseado
        AAFAuth-->>AuthActions: throw AuthError
        AuthActions->>Redux: dispatch(setAuthError)
        Redux-->>LoginUI: error state
        LoginUI-->>User: Muestra mensaje de error
    end
```
