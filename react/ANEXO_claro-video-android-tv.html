## 🔧 Troubleshooting

### Errores Comunes

#### 1. Build falla: JDK incorrecto

**Síntoma:** `Could not determine java version from '21.x.x'` o `Unsupported class file major version`

**Causa:** AGP 8.5.1 requiere exactamente JDK 17. JDK 11 y JDK 21 generan errores.

**Solución:**
```bash
java -version   # verificar que retorna 17.x.x
# En Android Studio: File вЖТ Project Structure вЖТ SDK Location вЖТ JDK 17
```

**Evidencia:** `gradle/wrapper/gradle-wrapper.properties` вАФ Gradle 8.11.1 requiere JDK 17+.

#### 2. SDK location not found / local.properties ausente

**Síntoma:** `SDK location not found. Define location with an ANDROID_SDK_ROOT environment variable`

**Causa:** `local.properties` no existe (está en `.gitignore`).

**Solución:** Crear `local.properties` en la raíz con `sdk.dir=<ruta>` (ver sección Configurar Archivos Secrets arriba).

**Evidencia:** `local.properties` verificado en repositorio; en `.gitignore`.

#### 3. Gradle sync falla por dependencia no encontrada

**Síntoma:** `Could not resolve com.google.dagger:hilt-android:2.44`

**Causa:** Versiones desactualizadas o de caché corrupta.

**Solución:**
```bash
./gradlew --stop
# Limpiar caché: eliminar ~/.gradle/caches/ (macOS/Linux) o %USERPROFILE%\.gradle\caches\ (Windows)
./gradlew dependencies
```

**Evidencia:** `app/build.gradle` вАФ dependencias con versiones estáticas declaradas.

- **Emulador lento:** Usar API 24, RAM mínima 2 GB, imagen `x86_64` para mejor rendimiento
