| **CWE** | CWE-319: Cleartext Transmission of Sensitive Information |

### Evidencia Directa

```swift
// Sources/CVAnalytics/Common/Traker/TrackerConstants.swift
struct TrackerConstants {
    static let baseURL = "http://www.google-analytics.com/collect"  // ❌ HTTP sin TLS
    // ...
}
```

### Datos que viajan sin cifrado

Los parámetros enviados a través de `GATracker` incluyen (según `TrackertParams` enum):
- `cid` — Client ID (identificador de sesión de usuario)
- `uid` — User ID
- `cd1`, `cd2`, `cd3` — Dimensiones personalizadas (pueden incluir datos de usuario)
- Información de dispositivo y plataforma

### Análisis de Impacto

**Riesgo técnico:**
- En redes WiFi públicas: los datos pueden ser interceptados sin herramienta especializada
- MitM (Man-in-the-Middle): trivial en HTTP claro
- iOS App Transport Security: require `NSAllowsArbitraryLoads` o excepción para `google-analytics.com` en la app host

**¿Está fallando en producción?**  
Si la app host NO tiene excepción ATS para `google-analytics.com`, **la petición HTTP es bloqueada silenciosamente por iOS**. En iOS 9+, ATS bloquea HTTP por defecto. `URLSession.dataTask` retorna error, pero `GATracker` no propaga el error al caller. Resultado: **analytics UA no enviados sin que el desarrollador lo sepa**.
